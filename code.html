This is my code.

I started coding in high school when I learned how to write progams in BASIC that would run on my TI-84 calculator. After that, I played around with a few languages in college, before realizing my love for code was was something I wanted to spend my life pursuing. I'm now working on becoming a full stack web developer. This site, along with all of the projects below, are all 100% my work, 100% written from scratch and, let's face it, 100% crap. Over time, I hope to turn the crap to gold, and hope this page will show that progression. It's also worth mentioning that I wrote the code for this entire site from scratch. No dreamweaver, just a simple text editor. The code is in decending order, by the date I finished it, so my newest work is at the top. Each title is a link to download the code file. Feel free to check out my work and let me know what you think. My email is on my info page.

    First real attempt at object oriented programming. This little game allows you to take care of a pet dragon.

  • 99 Bottles of Beer Lyrics, April 2015, Ruby

    This program prompts the user for a number of beers, then prints out the full lyrics to the '99 Bottles of Beer' song. Pretty simple, but I threw in a twist. I stole some code from my English Numbers program so that the lyrics use written out numbers rather than simply the numerical form.

  • English Numbers, April 2015, Ruby

    This program takes any number from 0 to 1,000,000 and outputs the number in English form. For example, one could enter 976,453 and the program would return nine hundred seventy-six thousand four hundred fifty-three.

  • Billion Seconds Birthday, April 2015, Ruby

    This program takes your birthday and tells you when you will (or did) turn one billion seconds old.
